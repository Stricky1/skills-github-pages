Oue é a tag NAV?

A tag NAV no HTML é usada para definir uma seção de navegação dentro de uma página.
Ela é especialmente útil para agrupar links de navegação, como menus principais, barras laterais ou qualquer conjunto de links que ajude os usuários a explorar o site.
